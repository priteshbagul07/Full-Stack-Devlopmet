# 🛍️ Assignment 3 – Fashion Store Website

This project is a responsive **Fashion Store Website** developed as part of a Full Stack Development assignment.
It combines HTML, CSS, and JavaScript to create an interactive shopping interface.

---

## 🚀 Features

* Attractive fashion product UI
* Responsive design layout
* Interactive elements using JavaScript
* Product display cards
* Hover effects and animations

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

```id="p3l9sd"
Assignment3.html   # Main HTML file
```

## 📄 Description

This project focuses on building a **fashion e-commerce interface** with interactive features.

The website includes:

* Product listing section
* Stylish UI with CSS effects
* JavaScript for dynamic behavior
* Clean and user-friendly layout

It helps in understanding how frontend technologies work together to build real-world applications.
