# 💬 Assignment 7 – Student Feedback System (MERN Stack)

This project is a **Student Feedback System** developed using the **MERN Stack (MongoDB, Express, React, Node.js)**.
It allows users to submit feedback and view feedback dynamically.

---

## 🚀 Features

* Submit student feedback
* Display feedback in real-time
* Full stack integration (Frontend + Backend)
* REST API communication
* Responsive UI using modern frontend tools

---

## 🛠️ Technologies Used

### Frontend:

* React.js (Vite)
* Tailwind CSS
* JavaScript

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB

## 📄 Description

This project demonstrates a **complete MERN stack application**.

The system:

* Allows users to submit feedback through a React frontend
* Sends data to the backend via API
* Stores feedback in MongoDB
* Displays feedback dynamically on the UI

It represents a real-world full-stack application using modern technologies.

## ▶️ How to Run

### 🔹 Backend Setup

1. Navigate to backend folder:

```id="b29xks"
cd backend
```

2. Install dependencies:

```id="z82kdl"
npm install
```

3. Start server:

```id="p82ksl"
node server.js
```

---

### 🔹 Frontend Setup

1. Navigate to frontend folder:

```id="k92lsl"
cd frontend
```

2. Install dependencies:

```id="d29xks"
npm install
```

3. Start frontend:

```id="l82kdl"
npm run dev
```

4. Open browser:

```id="p83kls"
http://localhost:5173
```
