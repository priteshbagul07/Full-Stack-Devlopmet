# 🏫 Assignment 5 – PCCOE Website (Node.js + EJS)

This project is a dynamic **College Website** developed using **Node.js, Express, and EJS** as part of a Full Stack Development assignment.
It demonstrates server-side rendering and routing.

---

## 🚀 Features

* Multi-page website using EJS templates
* Dynamic routing with Express.js
* Reusable layout components
* Static assets (images, CSS) handling
* Clean and structured UI

---

## 🛠️ Technologies Used

* Node.js
* Express.js
* EJS (Embedded JavaScript Templates)
* HTML5, CSS3

## 📄 Description

This project focuses on building a **dynamic college website** using backend technologies.

The application:

* Uses Express.js for routing
* Uses EJS for rendering dynamic pages
* Serves static files like images and CSS
* Organizes code in a structured MVC-like format

It helps in understanding how backend and frontend work together.

---

## ▶️ How to Run

1. Download or clone the repository

2. Open the project folder

3. Install dependencies:

```id="p92kdl"
npm install
```

4. Run the server:

```id="l29xks"
node app.js
```

5. Open browser and go to:

```id="z93ksl"
http://localhost:3000
```
