# 🌦️ Assignment 4 – Weather Dashboard

This project is a dynamic **Weather Dashboard Web Application** developed as part of a Full Stack Development assignment.
It fetches and displays real-time weather information using JavaScript and API integration.

---

## 🚀 Features

* Real-time weather data display
* Search weather by city name
* Dynamic UI updates using JavaScript
* Clean and responsive design
* Displays temperature, weather condition, and more

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Weather API (for real-time data)

---

## 📂 Project Structure

```id="w9x2ka"
Assignment4.html   # Main HTML file
Assignment4.css    # Styling file
Assignment4.js     # JavaScript logic (API integration)
```

---

## 📄 Description

This project focuses on building a **weather application** that interacts with an external API.

The application:

* Takes city input from the user
* Fetches weather data using an API
* Displays results dynamically on the webpage

It demonstrates how to work with APIs and update UI based on real-time data.

1. Download or clone the repository
