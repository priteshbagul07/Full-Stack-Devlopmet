# 🚀 Full Stack Development Assignments

This repository contains a collection of **7 Full Stack Development assignments**, demonstrating the progression from basic web development to advanced full-stack applications using modern technologies.

---

## 📌 Overview

The assignments are designed to build strong fundamentals and gradually move towards **real-world application development**.

* Starts with basic HTML
* Moves to CSS and JavaScript
* Covers API integration
* Introduces backend development (Node.js, Express)
* Ends with full-stack and MERN applications

---

## 🧩 Assignments Included

### 📘 Assignment 1 – Basic HTML Webpage

* Simple static webpage using HTML
* Focus on structure and basic tags

---

### 🎨 Assignment 2 – Personal Portfolio Website

* Portfolio UI using HTML & CSS
* Includes sections like About, Skills, Projects

---

### 🛍️ Assignment 3 – Fashion Store Website

* Interactive UI using HTML, CSS, JavaScript
* Product display with styling and effects

---

### 🌦️ Assignment 4 – Weather Dashboard

* Fetches real-time weather data using API
* Dynamic UI updates using JavaScript

---

### 🏫 Assignment 5 – PCCOE Website (Node.js + EJS)

* Backend project using Express and EJS
* Multi-page dynamic website

---

### 🚗 Assignment 6 – Vehicle Booking Application

* Full Stack project using Node.js, Express, MongoDB
* Features user authentication and booking system

---

### 💬 Assignment 7 – Student Feedback System (MERN Stack)

* MERN stack application
* React frontend + Node.js backend + MongoDB database
